Instructions:
  • Upload an image to enhance its contrast using the AGCWD algorithm.
  • Adjust the alpha value (α) to control the enhancement of the image.
  • Click the "Enhance" button to enhance the contrast of the input image.
  • Click the "Download" button to download the enhanced image.
  • Click the "Clear" button to clear the input image.
About:
  • The output image is a preview. Click the "Download" button to download the full image.
  • The algorithm used is the Adaptive Gamma Correction with Weighting Distribution (AGCWD) algorithm by Shih-Chia Huang, Fan-Chieh Cheng, & Yi-Sheng Chiu.
  • The implementation of AGCWD is from https://github.com/qyou/AGCWD/.
